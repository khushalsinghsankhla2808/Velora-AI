# Velora AI 🚀

An AI-powered website builder built using the MERN Stack. Velora AI aims to simplify website creation by combining modern web technologies with AI-driven workflows, providing users with an intuitive and efficient development experience.

## 📖 About The Project

Velora AI is a full-stack MERN application currently under active development. The project focuses on building a modern AI-powered platform that helps users create and manage websites efficiently.

The application follows industry-standard development practices, including frontend-backend separation, authentication, state management, and scalable project architecture.

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/khushalsinghsankhla2808/Velora-AI.git
cd Velora-AI
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the backend directory:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```

Create a `.env` file inside the frontend directory if required:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
```
